JavaScript

JavaScript adalah bahasa pemrograman yang digunakan developer untuk membuat halaman web yang interaktif. Dari menyegarkan umpan media sosial hingga menampilkan animasi dan peta interaktif,JavaScript digunakan terutama untuk membuat halaman web interaktif dan dinamis, seperti animasi, formulir, dan pembaruan konten secara real-time.

function greet(name) {

TS

TypeScript

TypeScript adalah salah satu cara populer untuk menambahkan definisi type ke dalam basis kode JavaScript. Secara bawaan, TypeScript mendukung JSX dan Anda bisa mendapatkan dukungan penuh React Web dengan menambahkan @types/react dan @types/react-dom ke proyek Anda.

interface User {
  id: number;
  name: string;
  email: string;
}

function createUser(user: User): User {
  return {
    id: Date().now(),
    ...user
  };
}

// Type safety at compile time
PY

Python

Python adalah bahasa pemrograman tingkat tinggi berorientasi objek yang ditafsirkan dengan semantik dinamis. Struktur data bawaan tingkat tinggi yang dipadukan dengan pengetikan dinamis dan pengikatan dinamis membuatnya sangat menarik untuk Pengembangan Aplikasi Cepat (RPA), serta untuk digunakan sebagai bahasa skrip atau bahasa perekat untuk menghubungkan komponen-komponen yang ada.

def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# Generate first 10 Fibonacci numbers
for i in range(10):
    print(fibonacci(i))
